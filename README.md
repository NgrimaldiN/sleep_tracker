# Sleep Tracker

Personal sleep-tracking app for logging Garmin sleep data and testing which habits seem to affect sleep quality.

## App

- Frontend: React + Vite
- Data store: Supabase
- Main UI: `src/`

## Root Layout

- App source lives in `src/` and `public/`
- Maintenance and one-off data scripts live in `scripts/maintenance/`
- Script-generated JSON state lives in `scripts/maintenance/logs/`
- Debug text dumps live in `artifacts/debug/`

## Commands

- `npm install`
- `npm run dev`
- `npm test`
- `npm run lint`
- `npm run build`
